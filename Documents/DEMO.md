# TestForge AI Demo Flow


## 1. Authentication

Register and login securely.

Features:

- User registration
- JWT Authentication
- Protected APIs



## 2. Dashboard

Displays:

- Projects
- Generated tests
- Executions
- Reports



## 3. Project Management

Create QA projects.

Example:

Project:

E-Commerce Automation Testing


Description:

AI powered QA automation for shopping cart, checkout, and payment workflows.



## 4. AI Test Generator


Input:

As a customer,
I want to add products to my cart,
update quantity,
remove products,
and complete checkout.


AI Flow:


Requirement

↓

Requirement Agent

↓

Multi LLM Orchestrator

↓

Gemini | Llama | DeepSeek

↓

Judge Agent

↓

Optimized Test Cases


Output:

- Positive tests
- Negative tests
- Security tests
- Test data



## 5. Test Execution

Runs automation scripts.

Provides:

- Execution status
- Logs
- Pass / Fail result



## 6. Bug Analyzer


Input:

Failed execution logs


Output:

- Severity
- Root cause
- Possible fix
- QA recommendation



## 7. Self Healing


Detects broken Selenium locators.

Automatically repairs test scripts.



## 8. CI/CD Generator


Generates:

- GitHub Actions pipeline
- Dependency installation
- Test execution workflow



## 9. Repository Intelligence


Analyzes GitHub repositories.

Finds:

- Tech stack
- Security issues
- Code improvements



## 10. Export Center


Exports generated QA reports.
